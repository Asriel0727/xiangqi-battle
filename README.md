## 如何移動？
1. 點擊表格中的位置連結 (如 `a2-a3`)
2. 系統會自動建立包含移動指令的 Issue
3. 直接提交該 Issue 即可完成移動  (若未更新棋盤，請稍等幾秒或是重新刷新頁面)
